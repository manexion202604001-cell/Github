# K2J Bridge — 韓国→日本 商品開発支援プラットフォーム

韓国企業が日本市場向けに新商品を開発するための支援プラットフォームのサイトです。
「日本でいま何が売れているか」のトレンド分析から、OEM商品企画・デザイン提案・マーケティングまでをワンストップで支援するプロダクトのコンセプトを、ランディングページ+操作できるデモダッシュボードとして実装しています。

既存の顧客台帳アプリと同じく **ビルド不要・静的HTMLのみ** の構成です。ブラウザで開くだけで動きます。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ランディングページ(サービス紹介・料金・FAQ) |
| `dashboard.html` | デモダッシュボード(4タブ・サンプルデータで動作) |

## デモダッシュボードの内容

| タブ | 内容 |
|---|---|
| 📊 トレンド分析 | **Qoo10(日本)をメインソース**としたカテゴリー別(美容・食品・生活雑貨)の急上昇ランキング、KPI、成長率グラフ、レビューAIインサイト |
| 💡 AI商品企画 | カテゴリー・価格帯・販路を選ぶと、OEM商品コンセプト+**商品イメージ画像(3案)**+規制チェック+工場マッチングを生成。**Claude APIキーを設定すると実際にAIがリアルタイム生成**(未設定時はサンプル表示) |
| 🎨 デザイン提案 | 日本市場向けデザイントンマナのムードボード。各ムードボードから**Pinterest・Behance・Dribbble・Qoo10の実際の商品/作品**を参照できる外部リンク付き |
| 🚀 マーケティング | 発売までの6ヶ月ロードマップ、販売チャネル適合度、ローンチKPIの目安 |

## Claude APIによるAI商品企画(BYOK方式)

「AI商品企画」タブでは、ユーザー自身の Anthropic APIキーを使って実際に Claude が商品コンセプトを生成できます。

- キー入力欄は `⚙️ Claude API設定` から開けます
- **APIキーはブラウザの `localStorage` にのみ保存され、Anthropic社のAPIへブラウザから直接送信されます。当サイト(GitHub Pages)のサーバーには一切送信・保存されません**
- API利用料はキーの持ち主(あなた)のAnthropicアカウントに発生します
- 生成プロンプトには「実在するブランド名・商品名と重複・類似させない、完全にオリジナルな名称にする」制約を明示しており、既存商品名との衝突を避けます
- APIキー未設定時、または呼び出しに失敗した場合は自動的にサンプルデータへフォールバックします

⚠️ **セキュリティ上の注意**: これは個人利用・プロトタイピング向けの簡易実装(BYOK: Bring Your Own Key)です。共有PCや複数人が使う環境では使用しないでください。不特定多数が使うサービスとして展開する場合は、APIキーをサーバー側(Supabase Edge Functions や Vercel Serverless Functions など)で管理し、フロントエンドに直接キーを渡さない構成に変更することを強く推奨します。

```
https://<アカウント名>.github.io/<リポジトリ名>/k2j-bridge/
```

## 注意

- トレンドランキング・数値・工場情報はすべて**説明用のサンプルデータ**です(実データ連携は今後の実装)
- 商品企画は、Claude APIキー未設定時は**サンプルの商品コンセプト**(VEILMER/HANMAEK/BREEZOという架空ブランド、実在商品とは無関係)を表示します
- 商品イメージ画像はSVGで描画したモックアップです(実際の商品写真ではありません)。実サービスでは写真品質の画像生成モデルへの置き換えを想定
- デザイン提案のPinterest/Behance/Dribbble/Qoo10リンクは、キーワード検索結果ページへの実際の外部リンクです(特定の商品・作品を保証するものではありません)
- 外部ライブラリ・CDNに依存せず、HTML/CSS/JSのみで完結しています
