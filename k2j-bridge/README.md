# K2J Bridge — 韓国→日本 商品開発支援プラットフォーム

韓国企業が日本市場向けに新商品を開発するための支援プラットフォームのサイトです。
「日本でいま何が売れているか」のトレンド分析から、OEM商品企画・デザイン提案・マーケティングまでをワンストップで支援するプロダクトのコンセプトを、ランディングページ+操作できるデモダッシュボードとして実装しています。

既存の顧客台帳アプリと同じく **ビルド不要・静的HTMLのみ** の構成です。ブラウザで開くだけで動きます。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ランディングページ(サービス紹介・料金・FAQ) |
| `dashboard.html` | デモダッシュボード(4タブ・サンプルデータで動作) |

## デモダッシュボードの内容

| タブ | 内容 |
|---|---|
| 📊 トレンド分析 | **国内主要ECモールをメインソース**としたカテゴリー別(美容・食品・生活雑貨)の急上昇ランキング、KPI、成長率グラフ、レビューAIインサイト |
| 💡 AI商品企画 | カテゴリー・価格帯・販路を選ぶと、OEM商品コンセプト+**商品イメージ画像(3案)**+規制チェック+工場マッチングを生成。**AI APIキーを設定すると実際にAIがリアルタイム生成**(未設定時はサンプル表示)。**収益シミュレーター・商標事前確認・見積依頼文コピー・提案書の印刷/PDF保存**などメーカー企業の実務に直結する機能を搭載 |
| 🎨 デザイン提案 | タイル状(マスonry)グリッドで実写真を表示。業界共通トレンドは**実在するデザイン参考画像**、**AI商品企画で生成した商品、または自社商品情報を読み込ませる**と**主要ECモールの実商品写真**(いずれも実画像検索設定時)。未設定時はCCライセンス素材で代用。各種検索結果への外部リンク付き |
| 🚀 マーケティング | 発売までの6ヶ月ロードマップ、販売チャネル適合度、ローンチKPIの目安 |

## AI APIによるAI商品企画(BYOK方式)

「AI商品企画」タブでは、ユーザー自身のAI APIキーを使って実際にAIが商品コンセプトを生成できます。

- キー入力欄は `⚙️ AI API設定` から開けます
- **APIキーはブラウザの `localStorage` にのみ保存され、AI APIの提供元へブラウザから直接送信されます。当サイトのサーバーには一切送信・保存されません**
- API利用料はキーの持ち主(あなた)のAPIアカウントに発生します
- 生成プロンプトには「実在するブランド名・商品名と重複・類似させない、完全にオリジナルな名称にする」制約を明示しており、既存商品名との衝突を避けます
- APIキー未設定時、または呼び出しに失敗した場合は自動的にサンプルデータへフォールバックします

⚠️ **セキュリティ上の注意**: これは個人利用・プロトタイピング向けの簡易実装(BYOK: Bring Your Own Key)です。共有PCや複数人が使う環境では使用しないでください。不特定多数が使うサービスとして展開する場合は、APIキーをサーバー側(Supabase Edge Functions や Vercel Serverless Functions など)で管理し、フロントエンドに直接キーを渡さない構成に変更することを強く推奨します。

## メーカー企業の実務に直結する機能(AI商品企画タブ)

生成された提案には、以下の実務機能が付属します。

- **💹 収益シミュレーター**: 想定売価・原価率・月間販売目標・月間広告費を入力すると、粗利/個・月間粗利・広告費控除後利益・損益分岐販売数をリアルタイム試算(簡易計算・税抜)
- **⚖️ 商標の事前確認**: AIが生成したブランド名は完全オリジナルですが、実際に使用する前に[特許庁の商標検索システム](https://www.j-platpat.inpit.go.jp/)・Web検索で商標の重複有無を確認できるリンクを用意
- **📋 見積依頼文のコピー**: マッチしたOEM/ODM工場ごとに、商品仕様・想定価格・確認事項を含む日本語の見積依頼文をワンクリックでクリップボードにコピー(実際の工場担当者へのメールにそのまま貼り付け可能)
- **🖨 提案書の印刷/PDF保存**: 生成した提案をそのまま印刷、またはブラウザの「PDFに保存」機能で一枚のPDF資料として出力し、社内共有・決裁に活用可能

## デザイン提案:実写真リファレンス

「デザイン提案」タブは、タイル状(マスonry)レイアウトで**実際の写真**を並べる参考資料ボードです。イラスト・モックアップではなく実物を参考にすることを意図しています。

- **業界共通のデザイントレンド(6ムードボード)**: **デザイン参考サイトに実在する画像**を表示(実画像検索設定時)
- **商品専用の参考写真**(AI商品企画で生成した商品/自社商品を読み込ませた場合): **主要ECモール・口コミサイトの実商品写真**を表示(実画像検索設定時)。実在の競合商品を参考にすることを意図

以下いずれかの方法で商品を読み込ませると、その商品専用の参考写真ボードを生成できます。

- **🤖 AI商品企画で生成した商品を読み込む**: 「AI商品企画」タブで生成した商品(直近10件、ブラウザに保存)から選択
- **✍️ 自社商品を入力する**: 商品名・カテゴリー・特徴キーワードを直接入力

生成される内容:
- デザインの方向性3案それぞれについて、実際の商品写真をマスonryグリッドで表示(AI APIキー設定時は方向性もAIが提案)
- 各写真には出典サイトを表示し、クリックで出典元ページへ
- 各案から実在するデザイン参考サイトの検索結果、作品集サイトの実例検索、ECモールの類似実商品検索へワンクリックで遷移可能

### 実画像検索の設定

「デザイン提案」タブの `⚙️ 実画像検索の設定` から、ご自身の**画像検索エンジン**のAPIキーと検索エンジンID(cx)を設定すると、業界共通のデザイントレンドはデザイン参考サイトに、商品専用の参考は主要ECモール・口コミサイトに絞り込んだ画像検索を行い、実際の画像を参考資料として表示します。

⚠️ これは外部サイトを独自にスクレイピングしているのではなく、**公式の画像検索API(利用規約準拠)**を通じて該当サイトの画像検索結果を取得しています。画像の著作権は各サイト・投稿者に帰属するため、当サイトでは画像を保存・再配布せず、その場で表示し出典元ページへのリンクを併記しています。

- **[① 検索エンジンを作成](https://programmablesearchengine.google.com/controlpanel/create)**(cxを取得)・**[② Custom Search APIキーを取得](https://console.cloud.google.com/apis/library/customsearch.googleapis.com)**
- キーはブラウザの`localStorage`にのみ保存され、検索APIの提供元へ直接送信されます(当サイトのサーバーには送信・保存されません)
- APIコンソールでAPIキーに**HTTPリファラー制限**を設定し、このサイトのURLのみを許可することを強く推奨します(AI APIキーと異なり、画像検索APIキーはこの制限を公式にサポートしています)
- 無料枠は1日100クエリまでです
- 未設定、または該当する実商品写真がない場合は、著作権フリー/クリエイティブ・コモンズライセンスの実写真を検索できるサービス(APIキー不要)に自動フォールバックします

## 注意

- トレンドランキング・数値・工場情報はすべて**説明用のサンプルデータ**です(実データ連携は今後の実装)
- 商品企画は、AI APIキー未設定時は**サンプルの商品コンセプト**(VEILMER/HANMAEK/BREEZOという架空ブランド、実在商品とは無関係)を表示します
- 「AI商品企画」タブの商品イメージ画像はSVGで描画したモックアップです(実際の商品写真ではありません)。実サービスでは写真品質の画像生成モデルへの置き換えを想定
- 「デザイン提案」タブの参考写真は、各外部サービス本体が実際にインデックスしている画像そのものではありません(公開APIがないため)。実画像検索設定時は主要ECモール等の実商品写真、未設定時はCCライセンス素材を表示します
- 各外部サイトへのリンクは、キーワード検索結果ページへの実際のリンクです(特定の画像・作品の存在を保証するものではありません)
- 収益シミュレーターは簡易計算です(税・送料・為替変動・返品率などは考慮していません)。実際の事業計画には専門家の確認を推奨します
- 商標事前確認リンクは検索の補助であり、商標登録の可否を保証するものではありません。実際の出願前には弁理士等の専門家にご確認ください
- 見積依頼文のコピー機能はテキスト生成のみです(サンプルの工場情報に実在の連絡先はありません)
- 外部ライブラリ・CDNに依存せず、HTML/CSS/JSのみで完結しています
