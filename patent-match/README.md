# PatentBridge — 特許×企業マッチングプラットフォーム

新しく開発した特許と、その技術を必要とする世界中の企業をマッチングするプラットフォームのサイトです。
既存のサイト群と同じく **ビルド不要・静的HTMLのみ** の構成で、ブラウザで開くだけで動きます。

## 解決する2つの課題

| 課題 | 解決策 |
|---|---|
| ① 特許は研究結果・請求項の羅列で、素人には「結局何ができる技術なのか」が分からない | **AI特許翻訳** — 一言要約/中学生にも分かる解説とたとえ話/従来技術との比較/ビジネス視点の価値、に自動変換 |
| ② その特許が「誰に・どんな風に」利用されるのかをマッチングするのが大変 | **用途展開エンジン+企業マッチング** — 技術の本質機能を抽出して異業種の活用シナリオを生成し、世界の企業と適合度スコア順でマッチング。提案文のAIドラフトまで支援 |

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ランディングページ(課題・ソリューション・使い方・料金・FAQ) |
| `dashboard.html` | デモダッシュボード(4タブ・サンプルデータで動作) |
| `universities.html` | 大学・研究機関向けページ(TLOの課題・大学向け機能・ROIシミュレータ・Academic料金) |
| `proposal.html` | 大学向け提案資料(スライド12枚・キーボード操作・「PDFとして保存」ボタンで印刷/PDF化可能) |

## デモダッシュボードの内容

サンプル特許3件(化学・材料/エレクトロニクス・ヘルスケア/バイオ・環境)を選ぶと、AI解析の流れを体験できます。テキスト貼り付け欄に特許文を貼ると、内容に近いサンプル解析が表示されます。

| タブ | 内容 |
|---|---|
| 💡 かんたん解説 | 一言要約/中学生にも分かる解説+たとえ話/従来技術との比較/ビジネス価値/本質機能タグ |
| 🧭 用途展開 | 本質機能をもとにした業界横断の活用シナリオ(市場規模感・実現ハードル・⭐意外な用途) |
| 🤝 企業マッチング | 世界の企業との適合度スコア・マッチング理由・企業ごとの活用シナリオ |
| ✉️ アプローチ | 企業別の提案文AIドラフト(日/英)、ライセンス・共同開発など連携形態のシミュレーション |

## 公開方法

このリポジトリをGitHub Pagesで公開している場合、そのまま以下のURLでアクセスできます。

```
https://<アカウント名>.github.io/<リポジトリ名>/patent-match/
```

## 注意

- 表示される特許・企業・スコア・金額はすべて**説明用の架空サンプルデータ**です(実在の特許・企業とは関係ありません)
- 外部ライブラリ・CDNに依存せず、HTML/CSS/JSのみで完結しています

## 今後の拡張候補

- AI解説・用途展開・提案文のLLM連携(Claude APIによるリアルタイム生成)
- 特許DB連携(J-PlatPat / Google Patents APIから特許番号で自動取得)
- 企業DB連携(企業の公開情報・ニーズ登録データベースの構築)
- 企業側の「技術ニーズ逆引き検索」画面
- Supabaseによるユーザー登録・特許保存・コンタクト管理(既存アプリと同じ構成で追加可能)
