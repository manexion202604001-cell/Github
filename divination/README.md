# 八珠の理(やたまのことわり)

八犬伝心相数 × 易経の多角鑑定占いアプリ。ビルド不要、`index.html` 1枚で動きます。

## 鑑定の構成

| 見立て | 内容 |
|---|---|
| 心相数 | 生年月日の数字をすべて足して一桁(または 11・22・33 のマスターナンバー)まで還元した数から、生まれ持った気質を読む |
| 八犬士の守護珠 | 南総里見八犬伝の八徳(仁・義・礼・智・忠・信・孝・悌)のうち、生まれ数が結ぶ珠と対応する犬士から、行動の指針を読む |
| 易経の卦 | 三枚の銭を六度投じる略筮法(乱数)で六十四卦から本卦を立てる。変爻があれば之卦(ゆくところの卦)も表示 |
| 八徳の均衡 | 八徳バランスをレーダーチャートで可視化 |
| 総合鑑定 | 三つの見立てを踏まえた総合の読みと、開運の色・方角 |

## 使い方

ブラウザで `index.html` を開くだけです。サーバーも API キーも不要で、入力内容はどこにも送信されません(すべてブラウザ内で計算)。

- **生年月日**(必須): 心相数・守護珠・八徳バランスの算出に使います
- **お名前**(任意): 八徳バランスと卦の種に混ぜられます
- **易に問いたいこと**(任意): 空欄なら本日の運気を占います

同じ人が同じ日に同じ問いを立てると同じ卦が出ます(人×問い×日付を種にした決定的な筮)。「卦を立て直す」で改めて筮を執れます。

## 公開する場合

このリポジトリの他サイトと同様、GitHub Pages / Vercel にそのまま置けば動きます。
