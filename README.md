# ❆ 古河めぐり — こがし観光スタンプラリー

茨城県古河市の観光スポット **12か所** を巡るデジタルスタンプラリーの Web アプリです。
GPS チェックインでスタンプを集め、特典ランクを獲得し、全制覇すると「完全制覇認定証」を発行できます。

土井利位の『雪華図説』にちなんだ **雪華模様 × 藍色** の和モダンデザインです。

## 🚀 動かし方

ビルド不要の静的サイトです。そのままホスティングするだけで動きます。

### ローカルで試す

```bash
# リポジトリ直下で
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

> 位置情報 API は `https://` または `localhost` でのみ動作します。
> `file://` で直接開いた場合、GPS チェックインは使えません（合言葉・デモモードは使えます）。

### GitHub Pages で公開する

リポジトリの **Settings → Pages** でこのブランチの `/`(root) を指定するだけで公開できます。

## 📱 主な機能

| 機能 | 説明 |
| --- | --- |
| GPS チェックイン | スポットの半径 200〜800m 以内で位置情報を使ってスタンプ取得（GPS 誤差は訪問者に有利に補正） |
| 合言葉チェックイン | GPS が使えない端末向け。現地掲示を想定した「合言葉」入力でスタンプ取得 |
| デモモード | 位置情報に関係なくチェックインを試せる動作確認用モード（ホーム画面の設定でON） |
| マップ | Leaflet + OpenStreetMap。取得済み/未取得でピンの色分け、現在地表示 |
| スタンプ帳 | 雪華ハンコ風スタンプ。押印アニメーション・取得日時つき |
| 特典ランク | 3個=🥉銅 / 6個=🥈銀 / 9個=🥇金 / 12個=👑完全制覇 |
| 完全制覇認定証 | 名前入りの認定証を Canvas で生成し PNG ダウンロード |
| シェア | Web Share API（未対応環境はクリップボードコピー） |
| PWA | ホーム画面に追加可能。Service Worker でアプリ本体をオフラインキャッシュ |

## 🗂 構成

```
├── index.html            # 画面（SPA・タブ切り替え）
├── css/style.css         # スタイル（雪華×藍の和モダン）
├── js/spots.js           # スポットデータ・特典定義（編集はここ）
├── js/app.js             # アプリ本体（チェックイン・スタンプ帳・地図など）
├── icons/icon.svg        # アプリアイコン
├── manifest.webmanifest  # PWA マニフェスト
└── sw.js                 # Service Worker（オフラインキャッシュ）
```

## 🔧 カスタマイズ

- **スポットの追加・変更**: `js/spots.js` の `SPOTS` 配列を編集します。
  `lat` / `lng`（座標）、`radius`（チェックイン許容半径 m）、`keyword`（合言葉）を現地に合わせて調整してください。
  ※ 同梱の座標は開発用のおおよその値です。実運用前に必ず現地で確認してください。
- **特典の変更**: `js/spots.js` の `REWARDS` 配列を編集します。
- スタンプの取得状況はブラウザの `localStorage` に保存されます（サーバー不要）。
